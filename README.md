# Parkinson Disease Detection Using Spiral Images

A deep learning project that detects Parkinson's disease from spiral drawings using Convolutional Neural Networks (CNN).

## Project Overview

Parkinson's disease affects motor control and often causes tremors. These tremors create distortions in spiral drawings. This project analyzes spiral images to automatically classify whether a drawing belongs to a healthy individual or a patient with Parkinson's disease.

## Features

- Image preprocessing using OpenCV
- Spiral image normalization
- CNN-based classification
- Automated prediction on new images
- Visualization of prediction confidence

  ## Technologies Used

- Python
- OpenCV
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Scikit-learn

## Dataset

The dataset contains spiral drawings from:
- Healthy individuals
- Parkinson's patients

Images are resized to 128×128 grayscale format.

## Model Architecture

- Convolutional Layers
- ReLU Activation
- MaxPooling Layers
- Fully Connected Layers
- Sigmoid/Softmax Output Layer

## Results

The model achieved high classification accuracy in distinguishing healthy and Parkinson-affected spiral drawings.

## Applications

- Early disease screening
- Medical decision support
- Low-cost diagnostic assistance
